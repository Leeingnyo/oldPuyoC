# Old Puyo C

![플레이 GIF](./oldPuyoC.gif)

새로 잘 만들기 위해 노력한 것은 [https://github.com/Leeingnyo/newPuyoC](https://github.com/Leeingnyo/newPuyoC)를 참고하세요.
